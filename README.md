# Go-Docker
 
 Infrastructura que contiene los docker images de GOCD (www.gocd.org) y los agentes en nodejs
 
Para iniciar los servicios es necesario tener instalado docker y luego ejecutar el siguiente comando.
 
 ```sh
 $ docker-compose up
 ```
 
 
 
### Link de GOCD Server
```sh
localhost:80/go
```
